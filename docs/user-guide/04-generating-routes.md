# Generating Routes

You can generate walking routes in any of your favorited regions.

## Opening the Route Planner

The Route Planner panel is available in the side panel when you have a favorited region
selected. It contains both the route generation form and a "Start Composing" button for
manual route composition.

## Setting a Distance

Use the distance slider to choose your target route distance (in kilometers). The
generated route will approximate this distance.

## Choosing a Route Type

- **One-way** - the route goes from a start point to a different end point
- **Loop** - the route returns to the starting point

## Setting Start and End Points (Optional)

You can optionally specify where your route should start and/or end:

1. Click "Pick on map" next to the start or end point field
2. Click a location on the map
3. Choose "Use Once" to use it for this generation only, or "Save as Place" to save it
   for future use
4. Alternatively, select a previously saved place from the dropdown

For loop routes, only the start point can be set (the route returns to the same point).

## Generating the Route

Click "Generate" to create your route. The route appears highlighted on the map. You
can:

- save the route for later
- clear it and generate a new one
- view the route distance and street names

If the distance between your chosen start and end points is longer than your requested
distance, the app will use the shortest possible path and notify you.
