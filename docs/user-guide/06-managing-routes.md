# Managing Routes

## Saved Routes

All routes you save (both generated and composed) appear in the "Saved Routes" section
of the side panel.

## Loading a Route

Click on a saved route to load it onto the map. The route is displayed with its full
geometry.

## Renaming a Route

Click the edit icon next to a route's name, type a new name, and confirm.

## Tracking Walked Routes

To record that you've walked a route, use the Walk History system. See
[Tracking Progress](07-tracking-progress.md) for details on adding walks from saved
routes, drawing on the map, or uploading GPX files.

## Deleting a Route

Click the delete icon on a saved route to permanently remove it. This action cannot be
undone.

## Exporting a Route

You can export any saved route as a file for use in other apps:

- **GPX** - compatible with most GPS and fitness apps

The export downloads a file to your device with the route's path.

## Route Limit

You can save up to 25 routes per region.
