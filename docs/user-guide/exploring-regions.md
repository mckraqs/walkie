# Exploring Regions

## The Explore Page

The explore page is the main screen of Walkie. It has:

- a header bar at the top with filters and controls
- a full-screen interactive map

## Filtering by District

Use the first dropdown in the header to filter regions by administrative district.
Select "All districts" to see every region.

## Selecting a Region

Use the second dropdown to choose a region. Regions are grouped into:

- **Favorites** - regions you have favorited (shown first)
- **Other regions** - all remaining regions

When you select a region, the map loads and displays all the streets and paths in that
area.

## Favoriting a Region

Click the star icon next to the region selector to add or remove a region from your
favorites.

Favoriting a region unlocks additional features:

- generating walking routes
- saving and managing routes
- creating places
- tracking walked progress

When you remove a region from favorites, all your saved routes and places in that region
are permanently deleted. You will be asked to confirm if you have any saved data.
